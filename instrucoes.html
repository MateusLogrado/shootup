Instruções

Neste jogo, você controla um personagem em meio a uma chuva de ataques inimigos. Seu objetivo é desviar dos tiros dos chefes enquanto ataca para derrotá-los e avançar para a próxima fase tendo até 3 bosses esse sendo o limite.

Controles

Setas do teclado → Movimentam o personagem.

X → Dispara tiros rápidos que causam dano pequeno.

C → Usa uma bomba poderosa, causando dano maior aos inimigos.
